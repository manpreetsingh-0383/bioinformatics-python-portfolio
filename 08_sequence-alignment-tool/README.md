# 🧬 DNA Sequence Alignment Tool  
### Needleman-Wunsch Algorithm (Global Alignment)

A Python-based bioinformatics tool that performs **global sequence alignment** using the Needleman-Wunsch algorithm.

---

## 🚀 Features

- 🔬 Aligns two DNA sequences
- ⚡ Uses dynamic programming
- 📊 Calculates optimal alignment score
- 🧬 Detects matches, mismatches, and gaps
- 💻 Clean and efficient Python implementation

---

## 🧠 Algorithm Used

This project uses the **Needleman-Wunsch Algorithm**, a dynamic programming technique used for optimal global sequence alignment.
