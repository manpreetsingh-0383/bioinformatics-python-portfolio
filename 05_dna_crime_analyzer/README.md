# DNA Crime Analyzer

This project matches a DNA sample from a crime scene with suspect DNA sequences.

## 🧬 Features
- Accepts crime scene DNA
- Compares with multiple suspects
- Identifies matching suspect

## ⚙️ How it Works
- DNA from the crime scene is compared with stored suspect DNA
- If a match is found, the suspect is identified

## 📌 Example
Crime DNA: ATGCGTAC  
Output: Suspect 1 is a MATCH!

## ▶️ How to Run
Run the script and enter the DNA sequence.

## 💡 Applications
- Forensic analysis
- DNA matching
