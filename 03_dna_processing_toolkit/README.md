# DNA Processing Toolkit

This project performs multiple DNA sequence operations.

## 🧬 Features
- Calculates DNA length
- Computes GC content
- Generates reverse sequence
- Finds complement
- Finds reverse complement

## ⚙️ How it Works
- GC content = (G + C) / Length × 100
- Reverse = sequence[::-1]
- Complement = base pairing (A-T, G-C)
- Reverse complement = complement + reverse

## 📌 Example
DNA: ATGC  
Reverse: CGTA  
Complement: TACG  
Reverse Complement: GCAT  

## ▶️ How to Run
Run the script and enter a DNA sequence.

## 💡 Applications
- Sequence analysis
- Bioinformatics preprocessing
