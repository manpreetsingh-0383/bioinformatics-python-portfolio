# DNA to Protein Translator

This project converts a DNA sequence into a protein sequence.

## 🧬 Features
- Converts DNA to RNA
- Translates RNA into protein sequence
- Uses codon mapping

## ⚙️ How it Works
1. DNA sequence is converted to RNA (T → U)
2. RNA is read in codons (3 bases)
3. Each codon is mapped to an amino acid

## 📌 Example
DNA: ATGGTT  
RNA: AUGGUU  
Protein: MV

## ▶️ How to Run
Run the script and enter a DNA sequence.

## 💡 Applications
- Protein synthesis study
- Genetic analysis
