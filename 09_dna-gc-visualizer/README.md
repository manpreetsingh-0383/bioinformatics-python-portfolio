# 🧬 DNA GC Content Analyzer

Not all DNA regions are the same…

Some parts are GC-rich while others are AT-rich — and this variation plays an important role in genome stability and gene expression.

This project is a simple Python-based tool that calculates and visualizes GC content across a DNA sequence using a sliding window approach.

---

## 🚀 Features

- Calculate GC content (%) of a DNA sequence
- Sliding window analysis for local GC variation
- Graphical visualization using matplotlib
- Beginner-friendly bioinformatics project

---

## 🧠 How It Works

1. Counts number of **G (Guanine)** and **C (Cytosine)** bases  
2. Calculates GC percentage:
   
   GC% = (G + C) / Total Bases × 100  

3. Applies **sliding window technique** to analyze variation across sequence  
4. Plots GC content using a graph
